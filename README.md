# Camden Cultural Preservation Society

Website for Sydney-based ambient drone collective.

**Live site:** [camdenculturalpreservationsociety.com](https://camdenculturalpreservationsociety.com)

## Stack

- Hosted on GitHub Pages
- DNS via Cloudflare
- Mailing list via Buttondown

## Files

```
├── index.html              # Main website
├── CNAME                   # Custom domain configuration
├── robots.txt              # Search engine directives
├── sitemap.xml             # Sitemap for SEO
├── llms.txt                # Context for AI assistants
├── ccps-icon.jpg           # Favicon
└── misery-tourist-artwork.jpg  # Social share image
```
